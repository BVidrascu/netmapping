Financial Network


Instructions


Every node in the network represents an actor involved in the implementation of the European Code Against Cancer. Their color shows the level they operate at (European, National or Regional), and their size is given by the perceived influence they have in the context of the implementation of the Code.


Every connection represents a financial relationship between two actors. The color of the tie signifies the level of the source and the recepient of the financial benefits (e.g. a green tie will be one between two actors at european level, while a teal one will be one between an actor at european level and one at national level, a combination of the two colors of the nodes that form the tie).


On the left of the screen you can find the main menu on the left, which you may close at any time. Within it, you have the option of searching for a certain node via the search function or selecting a group, which enables you to only see the nodes at one of the three levels.


By selecting any of the nodes, you will be able to only see the nodes connected to that particular node, as well as accessing the attributes panel. In this section, you will find more information about each of the nodes, as well as an interactive list of all the actors with which it interacts.



For the best experience, it is recommended that you view this page on desktop using Mozilla Firefox.
